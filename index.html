O que é um array?

Em JavaScript, uma "array" é como uma caixa de lápis, e cada lápis é um "elemento" dentro dela.

ForEach

Imagina que você quer dar um "Olá" para cada um dos seus elementos. O forEach() faz exatamente isso! Ele pega cada elemento, um por um, e nesse caso dá olá a eles.


Exemplo:
array.forEach(elem => {console.log(`Olá, ${elem}!`)})

Map

O map() é como uma máquina de fazer cópias especiais dos seus elementos. Ele pega cada elemento, faz uma cópia dele, mas com uma mudança que você escolhe. Existe uma variação chamada flatMap(), que funciona como o map(), mas com um passo extra. Primeiro, ele aplica uma função de transformação a cada item do array (como o map() faz), mas depois ele "achata" o resultado. Exemplo:


Exemplo:

Reduce

O método reduce() executa uma função reducer (fornecida por você) para cada elemento do array, resultando num único valor de retorno. Existe uma variação chamada reduceRight(), que funciona do mesmo jeito que o reduce(), mas começa do fim do array, indo do último item até o primeiro, em vez de começar pelo início.


Exemplo:
const soma = numeros.reduce((acumulador, valorAtual) => {return acumulador + valorAtual;}, 0);

Filter

O filter() é como um filtro mágico que você coloca na sua caixa de lápis. Ele olha para cada lápis e decide se ele deve ficar na caixa nova ou não, baseado em uma regra que você dá.


Exemplo:
const resultado = palavras.filter((palavra) => palavra.length > 6);

Every

O every() verifica se todos os itens de um array passam por uma condição. Se todos passarem, ele retorna true. Se apenas um falhar, retorna false.


Exemplo:

with()

O with() cria uma cópia do array, substituindo o valor em um índice específico. Ele não altera o original, e funciona de forma imutável. Requer suporte moderno do navegador (ES2023).


Exemplo:

Operador Spread (...)

O operador ... (spread) permite espalhar os elementos de um array em outro local — seja dentro de outro array, como argumentos de função ou em objetos.


Exemplo:

Operador Rest (...)

O operador ... (rest) junta múltiplos argumentos ou valores restantes em um array. Muito usado em funções com número indefinido de argumentos.


Exemplo:
const [a, ...resto] = [1, 2, 3, 4] // a = 1 e resto = [2, 3, 4]
