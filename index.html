O que é um array?

Em JavaScript, uma "array" é como uma caixa de lápis, e cada lápis é um "elemento" dentro dela.

ForEach

Imagina que você quer dar um "oi" para cada um dos seus lápis. O forEach() faz exatamente isso! Ele pega cada lápis, um por um, e faz alguma coisa com ele.


Exemplo:
array.forEach(elem => {console.log(`Olá, ${elem}!`)

Map

O map() é como uma máquina de fazer cópias especiais dos seus lápis. Ele pega cada lápis, faz uma cópia dele, mas com uma mudança que você escolhe.


Exemplo:
array.map(elem => elem.toUpperCase())

Reduce

O método reduce() executa uma função reducer (fornecida por você) para cada elemento do array, resultando num único valor de retorno.


Exemplo:
const soma = numeros.reduce((acumulador, valorAtual) => {return acumulador + valorAtual;}, 0);

Filter

O filter() é como um filtro mágico que você coloca na sua caixa de lápis. Ele olha para cada lápis e decide se ele deve ficar na caixa nova ou não, baseado em uma regra que você dá.


Exemplo:
const resultado = palavras.filter((palavra) => palavra.length > 6);
